Help - How to Use the S3 Bucket Status Checker

To use this tool, simply access the following endpoints:

  • /check/buckets/v1 - Returns the status of the S3 buckets.

Example: Navigate to /check/buckets/v1 to see the status codes of various S3 bucket names generated based on a pattern in console and it'll instert into MongoDB.

  • /check/buckets/db/v1 - Returns the status of the S3 buckets from the MongoDB.

Example: Navigate to /check/buckets/db/v1 to see the status codes of various S3 bucket names from the MongoDB.
